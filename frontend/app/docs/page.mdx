### <span id="getting-started" className="scroll-mt-28">Getting started</span>

      <ol>
        <li>Create a user from the <strong>Account</strong> page.</li>
        <li>Issue an API key and store it securely.</li>
        <li>Use the key with <code>X-API-Key</code> for protected backend routes.</li>
      </ol>

### <span id="public-pricing" className="scroll-mt-28">Public pricing</span>

      <ul>
        <li>Browse available tiers on <strong>Pricing</strong>.</li>
        <li>
          Run a tier estimate to preview configured included units and total micros-based
          pricing.
        </li>
        <li>
          Missing unit prices are called out in the UI so incomplete pricing configuration
          is visible.
        </li>
      </ul>
