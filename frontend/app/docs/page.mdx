## Docs

This documentation section is authored in MDX so new documentation pages can be added without hardcoding JSX placeholders.

### <span id="getting-started" className="scroll-mt-28">Getting started</span>

1. Create a user from the **Account** page.
2. Issue an API key and store it securely.
3. Use the key with `X-API-Key` for protected backend routes.

### <span id="public-pricing" className="scroll-mt-28">Public pricing</span>

- Browse available tiers on **Pricing**.
- Run a tier estimate to preview configured included units and total micros-based pricing.
- Missing unit prices are called out in the UI so incomplete pricing configuration is visible.

### <span id="api-quick-note" className="scroll-mt-28">API quick note</span>

For local backend development, ensure the frontend has `NEXT_PUBLIC_API_BASE_URL` configured. The frontend proxy routes depend on it.
